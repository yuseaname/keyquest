## Missions Summary

### Daily Missions (5)
- Type 200 words ? 100 XP + 50 gold
- Maintain 90% accuracy ? 150 XP + 75 gold
- Defeat 3 enemies ? 200 XP + 100 gold
- Achieve 25 combo ? 250 XP + 125 gold + card chance
- Reach 80 WPM ? 300 XP + 150 gold

### Weekly Missions (3)
- Type 1,000 words ? 500 XP + 250 gold + skin chance
- Defeat 3 bosses ? 1,000 XP + 500 gold + relic chance
- Complete 2 realms ? 750 XP + 375 gold

### Realm Missions (3)
- Fire Caverns (95% accuracy) ? Blaze Armor (guaranteed)
- Fire Caverns (<5 min) ? Pyraxis Crown (50% chance)
- Frost Vale (no damage) ? Frostbound Warrior + Glaciana's Tear (guaranteed)
- Shadow Library (100% accuracy) ? Void Stalker + Ancient Manuscript (guaranteed)

### Boss Missions (3)
- Defeat Pyraxis ? Inferno Lord card (guaranteed)
- Defeat Pyraxis (no damage) ? Inferno Robes + Crown Fragment (guaranteed)
- Defeat Glaciana ? Frost Queen card (guaranteed)

### Achievement Missions (5)
- Reach level 20 ? Cherry Blossom Priestess (guaranteed)
- Reach 100 WPM ? "Swiftfinger" title
- Achieve 100 combo ? "Combo King" title
- Defeat 10 bosses ? "Spellbreaker" title
- Collect all monster cards ? "Completionist" title

## Helper Functions

### Check if collectible is owned
```typescript
import { useOwnedCollectibles } from '@/store';

const ownedCollectibles = useOwnedCollectibles();
const hasItem = ownedCollectibles.some(c => c.collectibleId === 'blade-dancer-blaze');
```

### Get collectibles by type
```typescript
import { getCollectiblesByType } from '@/game/collectibles';

const skins = getCollectiblesByType('skin');
const cards = getCollectiblesByType('monster-card');
```

### Get collectibles by rarity
```typescript
import { getCollectiblesByRarity } from '@/game/collectibles';

const legendaries = getCollectiblesByRarity('legendary');
```

### Award collectible to player
```typescript
import { awardCollectible } from '@/services/database';

await awardCollectible(userId, 'blade-dancer-blaze', 'skin');
```

### Equip collectible
```typescript
import { useCollectiblesStore } from '@/store';

const { equipSkin } = useCollectiblesStore();
equipSkin('blade-dancer-blaze');
```
